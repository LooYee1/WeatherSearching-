
A simple website project for checking weather/简单的天气查询网站项目

通过http+css+js实现的简单天气查询项目,后端通过express来实现api的对接.

#使用注意事项
需要先获取api的公钥和私钥
本项目使用的是心知天气的api
https://www.seniverse.com/
